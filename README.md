# Data Science & Analytics Internship – Future Interns

## Overview
This repository contains the work completed as part of my Data Science & Analytics Internship with Future Interns. The internship focused on solving real-world business problems using data analysis, visualization, and insights.

## Tasks Completed

### Task 1: Business Sales Performance Analytics (Power BI)
- Analyzed sales data to understand revenue trends and business performance
- Created an interactive Power BI dashboard
- Identified top-performing regions, products, and categories

### Task 2: Customer Retention & Churn Analysis (Python)
- Performed data analysis using Python (Pandas, Seaborn, Matplotlib)
- Identified key factors affecting customer churn
- Visualized patterns related to contract type, tenure, and pricing

### Task 3: Marketing Funnel Analysis (Python + Power BI)
- Analyzed user flow across marketing funnel stages (Visitors → Leads → Conversions)
- Calculated conversion rates and identified drop-offs
- Compared performance across different marketing channels
- Built both Python visualizations and Power BI dashboard

## Tools & Technologies Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI
- Jupyter Notebook
- Excel

## Key Learnings
- Understanding business problems through data
- Data cleaning and exploratory data analysis (EDA)
- Building dashboards and visualizing insights
- Identifying patterns and making data-driven decisions

## Conclusion
This internship helped me gain hands-on experience in both data analysis and visualization. It strengthened my ability to work with real datasets and extract meaningful insights that can support business decisions.
