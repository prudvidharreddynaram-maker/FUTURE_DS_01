# Task 1: Business Sales Performance Analytics

## Description
In this task, I analyzed a business sales dataset using Power BI. The goal was to understand sales performance across different regions, categories, and products.

## Tools Used
- Power BI
- Excel Dataset

## What I Did
- Created an interactive dashboard in Power BI
- Analyzed sales trends over time
- Compared sales across regions
- Identified top-performing products and categories
- Calculated key metrics like total sales and profit

## Key Insights
- The West region had the highest sales performance
- Technology category contributed the most revenue
- Some products like Phones and Chairs performed better than others
- Sales showed variation across different time periods

## Conclusion
This task helped me understand how to use Power BI for data visualization and how to extract useful insights from business data.
