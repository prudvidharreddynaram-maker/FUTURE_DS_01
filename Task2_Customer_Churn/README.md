# Task 2: Customer Retention & Churn Analysis

## Description
In this task, I worked on customer churn data using Python. The objective was to understand why customers leave and identify patterns that affect customer retention.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## What I Did
- Loaded and cleaned the dataset using pandas
- Performed exploratory data analysis (EDA)
- Created visualizations to understand churn patterns
- Analyzed factors like contract type, monthly charges, and tenure

## Key Insights
- Customers with month-to-month contracts have higher churn
- Higher monthly charges are associated with increased churn
- Customers with shorter tenure are more likely to leave
- Some services like fiber optic show higher churn rates

## Conclusion
This task helped me understand how to analyze data using Python and how to identify important factors affecting customer behavior.
