# Task 3: Marketing Funnel Analysis

## Description
In this task, I worked on analyzing a marketing funnel to understand how users move through different stages such as visitors, leads, and conversions. The aim was to identify where users drop off and which marketing channels perform better.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI

## What I Did
- Created a dataset representing marketing funnel stages across different channels
- Performed analysis using Python to calculate conversion rates and drop-offs
- Visualized channel performance and overall funnel using graphs
- Built a Power BI dashboard to represent the funnel in a more interactive way

## Key Insights
- Google Ads generated the highest number of conversions
- Social Media showed higher drop-off between visitors and leads
- Email campaigns had lower reach but relatively stable conversions

## Conclusion
The analysis shows that improving early-stage engagement and focusing on high-performing channels can help increase overall conversions. Combining both Python and Power BI helped in better understanding and presenting the data.
